Advent Calendar Website
-----------------------
This package contains a ready-to-use advent calendar website (index.html) and 30 placeholder images
in the `images/` folder named `pic1.jpg` ... `pic30.jpg`.

What you should do:
1. Replace the placeholder images in images/ with your own images keeping the same filenames
   (pic1.jpg ... pic30.jpg).
2. Upload the entire folder to GitHub and enable GitHub Pages (use the repository root / branch 'main')
   or upload to any static host.

Features included:
- Responsive, mobile-optimized layout
- Flip animation for doors
- Sidebar preview with download button
- Progress saved in localStorage
- Optional "Lock by date" checkbox (blocks opening future days based on current date)
- 30 generated placeholder images (if environment supports image generation)

To test locally: open index.html in a modern browser.
